数据库分类

关系类型数据库

  • 通过表和表之间，行和列之间的关系进行数据的存储
  • 常见的MySQL、Oracle、DB2

非关系型数据库

  • 非关系型型数据库，对象存储，通过对象自身的属性来决定。
  • 常见的有Redis、MongDB

DBMS（数据管理系统）

  • 数据库管理软件，科学有效的管理数据、维护和获取数据；
  • Mysql，数据库管理系统
  • 5.7版本属于比较稳定版本
  • 8.0版本驱动与5.7不一样，所以有时候需要配置一些不同的配置

SQL语句

数据库相关的SQL
    SHOW DATABASES;        --查看所有数据库
    CREATE DATABASE shool;    --创建shool数据库，shool是数据库名
    SHOW CREATE DATABASE shool    --查看shool数据库详情
    DROP DATABASE shool;    --删除shool数据库
    USE shool;    --使用shool数据库

表相关的
    CREATE TABLE ti (name VARCHAR(255),age INT) CHARSET=UTF8;    --创建表
    SHOW TABLES;    --展示所有表
    SHOW CREATE TABLES t1;    --查看t1表详情
    DESC ti;    --查看t1表字段
    DROP TABLE ti;    --删除t1表
    REMANE TABLE ti TO t2;    --将t1表改名为t2表

    ALTER TABLE t1 ADD id INT;    --添加字段id到表t1的最后面
    ALTER TABLE t1 ADD id INT FIRST;    --添加id字段到t1表的最前面 
    ALTER TABLE t1 ADD id INT AFTER name;    --添加id字段到name字段的后面
    ALTER TABLE t1 DROP id;        --删除t1表的id字段
    ALTER TABLE t1 CHANGE id uid INT;    --将t1表表字段id修改成uid，注意新名后面记得跟新类型

  • mediumint 中等大小的数据 3个字节
